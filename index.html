À propos de RblxDash
Dashboard non-officiel — non affilié à Roblox Corporation
Confidentialité totale
Aucune donnée n'est envoyée à nos serveurs. Tout reste dans ton navigateur.
API Roblox publique
Utilise uniquement les APIs publiques de Roblox. Aucun cookie requis.
Open Source
Code disponible sur GitHub. Contribue ou fork librement.
Limites connues de l'API
Les revenus Robux ne sont pas accessibles via l'API publique (nécessite une authentification Roblox Studio).
Les jeux doivent être publics pour afficher leurs statistiques.
La liste des amis est limitée à 200 entrées par requête.
